# Simple booking application built with React and Node.js microservices. Microservices communicates throught the Nats streaming module.  
# Demo: http://ticketing.pashkoostap.cloud.okteto.net/

# Used technologies
- Kuberenetes
- Docker
- Mongodb
- NATS Streaming
- Node.js
- Stripe
- Express
- React
- Jest
- GitHub actions
- Okteto cloud
